== Description ==

The GNU Compiler Collection, including the C, C++ and Fortran compiler.

== Upstream Contact ==

http://gcc.gnu.org/

== Dependencies ==

 * zlib
 * MPIR
 * MPFR
 * MPC

== Special Update/Build Instructions ==

* Lots of stuff can be removed, run ./spkg-src to do this automatically
  (after downloading the sources).
